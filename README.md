# 🤖 ChatBot com IA usando Streamlit e OpenAI

Aplicação web de chatbot com inteligência artificial desenvolvida em Python utilizando Streamlit para interface e OpenAI para geração de respostas.(obs:keyapi:"Inativa")

---

## 📌 Objetivo

Criar um chatbot interativo capaz de conversar com o usuário em tempo real, mantendo o histórico das mensagens e gerando respostas inteligentes com IA.

---

## 🚀 Tecnologias Utilizadas

- Python
- Streamlit
- OpenAI API
- Session State (armazenamento de histórico)

---

## 🧠 Funcionalidades

- Interface de chat interativa
- Envio de mensagens em tempo real
- Respostas geradas por IA
- Histórico de conversa persistente na sessão
- Estrutura simples e escalável
